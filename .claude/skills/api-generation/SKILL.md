# API generation

This Skill provides the commands to run backend tests, to verify functionality.

## Instructions

1. Go to the project root
2. run `./gradlew cleanGenerated`
3. run `./gradlew generateOpenApiDocs`
4. run `./gradlew generateTypeScriptClient`

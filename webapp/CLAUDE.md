# Project Guidelines

When working on this projects, we prefer readability and simplicity over cleverness. Always ask for input before
proceeding with big, impactful changes.

# Inspiration

Design inspiration comes from the webpage www.enode.com

## Standards

- SCSS for styling
- Angular as a frontend framework
- Reusability where possible
- Tidy codebase

## Common Commands

```bash
ng serve  # dev server, accessible via localhost:4200
```
